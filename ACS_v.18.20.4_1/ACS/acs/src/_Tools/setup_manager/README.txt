ACS Silent install/uninstall
----------------------------

On Windows:

1. Open a terminal as Administrator
2. Go to acs_setup_manager folder
3-1. Execute acs_silent_install.bat for installation
  or
3-2. Execute acs_silent_uninstall.bat for uninstallation

On Linux:

1. Open a terminal
2. Go to acs_setup_manager folder
3-1. Execute acs_silent_install.sh using sudo rights for installation
  or
3-2. Execute acs_silent_uninstall.sh using sudo rights for uninstallation
